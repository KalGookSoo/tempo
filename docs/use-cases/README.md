# Tempo 사용자 유즈케이스

이 디렉터리는 tempo MVP의 기능별 사용자 유즈케이스를 정리한다.

Mermaid 다이어그램은 라이트 모드와 다크 모드에서 모두 읽기 쉽도록 고대비 색상만 사용한다.

## 공통 Mermaid 색상 기준

- 기본 노드: 흰 배경, 검은 텍스트, 검은 선
- 사용자 액션: 연한 노란 배경, 검은 텍스트, 검은 선
- 종료/완료: 연한 초록 배경, 검은 텍스트, 검은 선
- 오류/취소: 연한 빨강 배경, 검은 텍스트, 검은 선
