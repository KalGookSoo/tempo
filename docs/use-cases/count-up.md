# 카운트업 유즈케이스

## 목적

사용자는 `00:00`에서 시작해 설정한 종료 시간까지 증가하는 타이머를 실행한다.

## 주요 사용자

- 개인 운동자
- 코치
- 시간 제한 운동을 진행하는 사용자

## 선행 조건

- 사용자는 카운트업 모드를 선택할 수 있다.
- 사용자는 종료 시간을 `00:00`부터 `99:59` 사이로 입력할 수 있다.

## 기본 흐름

1. 사용자가 카운트업 모드를 선택한다.
2. 사용자가 종료 시간을 입력한다.
3. 사용자가 준비 카운트다운과 알림 큐 설정을 확인한다.
4. 사용자가 시작 버튼을 누른다.
5. 준비 카운트다운이 켜져 있으면 준비 단계가 실행된다.
6. 타이머가 `00:00`에서 시작해 증가한다.
7. 설정한 종료 시간에 도달하면 멈춘다.
8. 종료 시 알림 큐가 실행된다.

## 대안 흐름

- 사용자는 실행 중 일시정지할 수 있다.
- 사용자는 일시정지 상태에서 재개할 수 있다.
- 사용자는 실행 중 리셋할 수 있다.
- 종료 시간이 `00:00`이면 시작할 수 없도록 안내한다.

- 항상 `00:00`에서 시작한다.
